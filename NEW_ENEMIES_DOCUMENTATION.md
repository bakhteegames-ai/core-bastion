# Новые типы врагов - Документация

## Обзор
Добавлено **7 новых уникальных типов врагов** для увеличения глубины геймплея и стратегического разнообразия Core Bastion.

---

## Новые типы врагов

### 1. Stealth (Невидимка) 🥷
- **HP:** 12 (+4 за волну)
- **Скорость:** 2.8
- **Награда:** 35 золота
- **Особенность:** Невидимость

**Механика:**
- Начинает волну невидимым (прозрачность 30%)
- Становится видимым только когда HP падает ниже 30%
- Требует от игрока угадывать позицию или использовать AoE способности

**Стратегия:** Используйте башни с уроном по области или замедлением

---

### 2. Healer (Целитель) 💚
- **HP:** 25 (+6 за волну)
- **Скорость:** 1.5
- **Награда:** 45 золота
- **Особенность:** Лечение союзников

**Механика:**
- Каждые 2 секунды лечит ближайших союзников на 3 HP
- Радиус лечения: 4 единицы
- Приоритетная цель для уничтожения

**Стратегия:** Убивайте первым, используйте снайперские башни

---

### 3. Spawner (Размножитель) 🥚
- **HP:** 35 (+8 за волну)
- **Скорость:** 1.3
- **Награда:** 55 золота
- **Особенность:** Призыв миньонов

**Механика:**
- Каждые 4 секунды призывает 1 Grunt миньона
- Максимум 3 миньона одновременно
- Миньоны появляются рядом со Spawner'ом

**Стратегия:** Контроль толпы, AoE урон

---

### 4. Splitter (Делитель) 🔮
- **HP:** 30 (+7 за волну)
- **Скорость:** 2.0
- **Награда:** 40 золота
- **Особенность:** Разделение при смерти

**Механика:**
- При смерти разделяется на 2 меньших врагов
- Дочерние враги имеют 40% HP родителя
- Может создать цепную реакцию

**Стратегия:** Добивайте быстро, не допускайте размножения

---

### 5. Speedster (Спринтер) ⚡
- **HP:** 6 (+2 за волну)
- **Скорость:** 5.0 (базовая)
- **Награда:** 25 золота
- **Особенность:** Рывки

**Механика:**
- Базовая скорость в 2.5x выше обычной
- С шансом 30% делает рывок (ускорение x2.5 на 1.5 сек)
- Очень опасен на поздних волнах

**Стратегия:** Замедление обязательно, приоритетная цель

---

### 6. Juggernaut (Джаггернаут) 🛡️
- **HP:** 120 (+20 за волну)
- **Скорость:** 0.8
- **Награда:** 100 золота
- **Особенность:** Тяжелая броня

**Механика:**
- Броня: 15 единиц
- Снижение получаемого урона на 50%
- Очень медленный, но живучий

**Стратегия:** Бронепробивающие башни, фокусированный огонь

---

### 7. Boss (Босс) 👹
- **HP:** 500 (+50 за волну)
- **Скорость:** 1.0
- **Награда:** 200 золота
- **Особенность:** 3 ультимативные способности

**Механика:**
- **Heal (70% HP):** Восстанавливает 10% здоровья
- **Summon (50% HP):** Призывает 2-3 Grunt миньона
- **Shield (30% HP):** Щит на 50% снижение урона

**Стратегия:** Готовьтесь к волне, имейте запас золота для апгрейдов

## Прогрессия волн

### Ранние волны (1-9)
- Только базовые враги: Grunt, Runner, Tank
- Обучение механикам

### Волна 10 - Первый Босс
- Boss + 4 Grunt + 2 Runner
- Проверка подготовки игрока

### Волны 11-14
- Вводится **Stealth** (с волны 12)
- Смесь старых и новых типов

### Волна 15
- **Healer + Spawner** дебют
- 5 Grunt + 2 Tank для поддержки

### Волны 16-19
- Комбинации всех специальных врагов
- Требуется стратегическое мышление

### Волна 20 - Второй Босс
- Boss + Juggernaut + 2 Healer + 2 Spawner + 5 Grunt
- Серьезный вызов

### Волны 21+
- Все типы врагов включая редких:
  - **Splitter** (с волны 22)
  - **Speedster** (с волны 23)
  - **Juggernaut** (с волны 24)
- Босс каждые 10 волн

## Интеграция с существующими системами

### EconomyService
- Новые награды за убийство сбалансированы
- Juggernaut дает больше золота за сложность

### UltimateAbilitySystem
- Stealth требует умного использования AoE ульта
- Speedster может быть контрен Meteor Strike
- Healer/Spawner идеальны для Lightning Storm

### Tower Types
| Башня | Эффективна против | Контрмеры |
|-------|------------------|-----------|
| Archer | Flyer, Speedster | Stealth |
| Cannon | Tank, Juggernaut | Healer |
| Ice | Speedster, Runner | Boss (щит) |
| Sniper | Healer, Spawner | Splitter |
| Lightning | Группы, Spawner | - |

## Технические детали реализации

### EnemyAgent.js
- Добавлены состояния для специальных способностей
- `_isVisible` для Stealth
- `_isDashing` для Speedster
- `_lastHealTime`, `_lastSpawnTime` для таймеров
- Метод `_updateSpecialAbilities(dt)` обновляет способности
- Метод `_splitEnemy()` обрабатывает разделение

### EnemySpawner.js
- Обновлен `_handleBossSummon()` для поддержки customHP и spawnPosition
- Поддержка спавна миньонов в указанной позиции

### enemyTypes.js
- 7 новых типов врагов с полными характеристиками
- Функция `getEnemyComposition()` с прогрессией по волнам
- Балансные параметры для каждого типа

---

## Баланс и настройки

Все параметры можно настроить в `/src/data/enemyTypes.js`:
```javascript
STEALTH: {
  baseHP: 12,        // Базовое здоровье
  hpPerWave: 4,      // Прирост за волну
  speed: 2.8,        // Скорость движения
  goldReward: 35,    // Награда
  invisibilityThreshold: 0.3 // Порог видимости (30% HP)
}
```

## Рекомендации для игроков

1. **Ранняя игра (волны 1-10):** Фокусируйтесь на экономике, стройте 2-3 башни среднего уровня
2. **Мид гейм (11-20):** Добавьте специализированные башни под новые типы врагов
3. **Поздняя игра (21+):** Максимальные апгрейды, используйте ульты стратегически

### Приоритеты целей:
1. Healer (лечит других)
2. Spawner (создает угрозу)
3. Speedster (быстро достигает базы)
4. Splitter (размножается)
5. Juggernaut (медленный, но-tanky)
6. Stealth (угроза, но низкий DPS)

## Визуальные эффекты

Каждый тип врага имеет уникальный визуальный стиль:
- **Stealth:** Полупрозрачный синий кристалл
- **Healer:** Зеленое свечение, пульсирующий эффект
- **Spawner:** Оранжевые наросты на корпусе
- **Splitter:** Фиолетовый цвет с трещинами
- **Speedster:** Желтые молнии на корпусе
- **Juggernaut:** Темно-серый с металлическими пластинами

Анимации включают:
- Пульсацию ядра
- Покачивание при движении
- Специфичные эффекты (крылья у Flyer, щит у Boss)
