# Ultimate Ability System - Документация

## Обзор

Система ультимативных способностей добавляет глубину геймплею Core Bastion, предоставляя игроку мощные инструменты для влияния на ход битвы.

## Новые способности

### 1. Meteor Shower (Метеоритный дождь) ☄️
- **Стоимость:** 250 золота
- **Перезарядка:** 75 секунд
- **Эффект:** Вызывает 5 метеоров, наносящих 80 урона по области радиусом 3м
- **Тактика:** Идеально для плотных волн врагов

### 2. Time Warp (Искажение времени) ⏰
- **Стоимость:** 175 золота
- **Перезарядка:** 60 секунд
- **Эффект:** Замедляет ВСЕХ врагов на 70% на 6 секунд
- **Тактика:** Спасение в критический момент, когда врагов слишком много

### 3. Chain Lightning (Цепная молния) ⚡
- **Стоимость:** 125 золота
- **Перезарядка:** 35 секунд
- **Эффект:** Молния бьет по 10 врагам, нанося 60 урона каждому
- **Тактика:** Быстрое уничтожение разрозненных групп

### 4. Tactical Nuke (Тактическая ядерка) ☢️
- **Стоимость:** 500 золота
- **Перезарядка:** 180 секунд
- **Эффект:** МГНОВЕННО уничтожает ВСЕХ врагов на экране
- **Тактика:** "Кнопка паники" для спасения проигранной игры

## Улучшенные классические способности

### Air Strike (Авиаудар) 💣
- **Улучшения:** Увеличен урон до 150, радиус до 4м
- **Визуал:** Добавлены частицы взрыва

### Absolute Zero (Абсолютный ноль) ❄️
- **Улучшения:** Полная заморозка вместо замедления, длительность 4 сек
- **Визуал:** Ледяная волна по всем врагам

### Divine Shield (Божественный щит) 🛡️
- **Улучшения:** Лечение + временный щит на 3 HP
- **Визуал:** Целебная аура вокруг базы

### Midas Touch (Прикосновение Мидаса) 👑
- **Улучшения:** 2x золота на 15 секунд (было 10 сек)
- **Визуал:** Золотая аура

## Интеграция с игрой

### Управление
- **Клавиши 1-4:** Классические способности
- **Клавиши 5-8:** Ультимативные способности
- **Клик по иконке:** Активация способности

### Визуальные эффекты
Каждая способность имеет уникальные VFX:
- Частицы разных цветов
- Длительность анимации
- Масштаб эффектов

### Звуковое сопровождение
- `explosion_large` - Авиаудар
- `ice_shatter` - Заморозка
- `holy_light` - Лечение
- `coins_rain` - Золотая лихорадка
- `meteor_impact` - Метеоры
- `time_warp` - Искажение времени
- `lightning_strike` - Молния
- `nuke_detonation` - Ядерный взрыв

## Баланс

### Стоимость vs Эффективность
| Способность | Cost/Efficiency | Cooldown |
|-------------|-----------------|----------|
| Meteor      | Высокая         | Средний  |
| Time Warp   | Средняя         | Средний  |
| Lightning   | Низкая          | Короткий |
| Nuke        | Очень высокая   | Длинный  |

### Стратегии использования
1. **Ранняя игра:** Lightning для фарма золота
2. **Средняя игра:** Meteor для контроля волн
3. **Поздняя игра:** Nuke как страховка
4. **Экономика:** Midas Touch между сложными волнами

## Технические детали

### Архитектура
```
UltimateAbilitySystem
├── ULTIMATE_ABILITIES (конфигурация)
├── ActiveAbilityEffects (трекинг эффектов)
└── UltimateAbilitySystem (ядро)
    ├── useAbility()
    ├── _executeAbility()
    ├── update()
    └── getGoldMultiplier()
```

### Интеграция с EconomyService
```javascript
// Gold multiplier применяется автоматически
economyService.addGold(baseAmount); // С учетом множителя
```

### Интеграция с WaveManager
```javascript
// Area damage и slow эффекты
waveManager.dealDamageInRadius(position, radius, damage);
waveManager.applySlowInRadius(position, radius, factor, duration);
```

## Будущие улучшения

1. **Дерево прокачки способностей**
   - Уменьшение кулдауна
   - Увеличение эффективности
   - Снижение стоимости

2. **Комбо-способности**
   - Freeze + Meteor = Massive damage
   - Time Warp + Lightning = Guaranteed hits

3. **Ежедневные испытания**
   - Ограниченный набор способностей
   - Бонусы за использование определенных комбинаций

4. **Достижения**
   - "Meteor Master" - Убить 100 врагов метеорами
   - "Nuclear Winner" - Использовать ядерку и выиграть волну

## Заключение

Система ультимативных способностей превращает Core Bastion из простой Tower Defense в глубокую стратегическую игру с элементами action. Каждая способность имеет свою нишу и требует правильного тайминга для максимального эффекта.
